# 🚀 TechFix – Repair & Service Management System

> A modern web-based platform to manage repair services, customers, and technical operations efficiently.

---

## 📌 Overview

TechFix is a full-stack web application designed to streamline the workflow of a tech repair/service business.  
It helps manage customers, repair requests, products, and administrative operations in one centralized system.

---

## ✨ Features

- 🔐 Authentication & Authorization  
- 🛠️ Repair Management  
- 📦 Product Management  
- 🧾 Customer Management  
- 🛒 Order / Service Handling  
- 📊 Admin Dashboard  

---

## 🏗️ Tech Stack

Frontend: HTML, CSS, JavaScript  
Backend: Java (JSP/Servlets), Tomcat  
Database: MySQL
